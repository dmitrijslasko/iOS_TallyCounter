# iOS TallyCounter
Simple Tally Counter iOS app to explore the joy of iOS development

**Current version: v1.0.0**

**Features:**
- Selection of counter colors
- Custom step size
- Reverse count
- Selection of tap sounds
- Selection of haptic feedback modes
  
**Gestures:**
- Drag up / down (increment / decrement)
- Long-press counter reset
- Shake (counter reset)
